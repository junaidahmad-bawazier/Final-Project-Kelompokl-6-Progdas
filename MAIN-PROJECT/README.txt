== Sistem Zonasi dan Rekomendasi Diversifikasi UMKM Retail ==

Program ini membantu para pelaku usaha kecil atau calon pengusaha UMKM menentukan lokasi bisnis yang strategis dan tidak jenuh. Program ini bekerja sebagai alat analisis yang menghitung kapasitas maksimal sebuah kategori usaha (seperti kuliner, sembako, jasa, hingga elektronik) berdasarkan jumlah penduduk di kawasan wilayah terpilih.


Fitur utama dan alur kerja program ini meliputi:

- Analisis Zonasi: Menentukan apakah sebuah wilayah masih memiliki kuota untuk jenis usaha tertentu atau sudah mencapai titik jenuh.
- Diversifikasi Usaha: Jika sebuah usaha dinyatakan sudah jenuh, program akan mendorong pengguna untuk melihat tabel potensi usaha lainnya yang sisa kuotanya masih banyak, sehingga persaingan bisnis tetap sehat.
- Sistem Rekomendasi: Mengurutkan wilayah dan jenis usaha berdasarkan sisa kuota tertinggi menggunakan algoritma sorting kemudian menampilkannya dalam tabel, memberikan saran terbaik bagi pengguna untuk membuka cabang atau usaha baru.
